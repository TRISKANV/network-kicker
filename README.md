# Network Kicker

Un script simple de ARP Spoofing para desconectar un dispositivo específico de una red local.

## ⚠️ Advertencia

Este script es para fines educativos. Usarlo en redes a las que no tenés permiso es ilegal y éticamente incorrecto. Úsalo bajo tu propio riesgo.

## Requisitos

*   Python 3
*   Biblioteca `scapy`
